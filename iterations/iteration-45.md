### 9. Abando as Indigenous vs. Basque Name — Updated Analysis

The Ancestry Philippines Select Marriages results show "Abando" and "Abandio" used as FIRST names in early records (pre-1849):
- **Abando Densing** (3 records) — "Abando" as given name, "Densing" as surname
- **Abando Acang** — "Abando" as given name
- **Abandio Raburar** — variant as given name
- **Abandio Tena** — variant as given name

This strongly suggests "Abando" was originally an **indigenous Filipino given name** that was adopted as a surname during or after the 1849 Catálogo de Apellidos — NOT a direct Spanish/Basque surname import. The Basque district of Abando (Bilbao) may be a false cognate.

This pattern is exactly like "Tamondong" — an indigenous name that became a surname in 1849.
